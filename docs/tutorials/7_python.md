## build venv

if you have conda, you could not use venv. first practice in [sphinx](https://www.sphinx-doc.org/en/master/tutorial/getting-started.html)

```bash
python -m venv ~/.venv

source ~/.venv/bin/activate

(.venv)$ python -m pip install sphinx
```
